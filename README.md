# Amazon_clone
Création d'un clone Amazon:
- Authentification (Sign Up, Sign In).
- Home.
- Panier.
- Paiement.

## Pour lancer l'application
`npm i && npm start`  

## Technologies utilisées
* React
* CSS3
* Material UI
* Firebase
* Axios
* Stripe
